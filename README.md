# 🛰️ NetScope Pro

[![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)](https://dart.dev)
[![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20iOS%20%7C%20Web-blue?style=for-the-badge)](https://flutter.dev)

**NetScope Pro** is a cutting-edge network diagnostic tool designed for developers, network engineers, and tech enthusiasts. It transforms traditional command-line network analysis into a beautiful, interactive, and visually rich experience.



## ✨ Key Features

- **🌐 Graphical Traceroute:** Visualize the path of your packets across the globe with interactive map integration.
- **📈 Real-time Analytics:** High-performance charts showing latency, jitter, and response times.
- **📍 Server Geolocation:** Automatically resolve and display the physical location of every network hop.
- **🎨 Premium UI/UX:** A modern interface built with fluid animations, glassmorphism, and dynamic transitions.
- **🌗 Adaptive Themes:** Seamless switching between sleek Dark Mode and polished Light Mode.
- **📜 Smart History:** Keep track of all your previous diagnostic sessions for quick comparisons.
- **📤 Share & Export:** Export detailed network reports or share path visualizations with your team.

## 🛠️ Tech Stack

- **Framework:** [Flutter](https://flutter.dev) - UI toolkit for building natively compiled applications.
- **State Management:** [Provider](https://pub.dev/packages/provider) - Simple and robust scalable state handling.
- **Visualization:** [fl_chart](https://pub.dev/packages/fl_chart) - Powerful Flutter chart library.
- **Maps:** [flutter_map](https://pub.dev/packages/flutter_map) + [latlong2](https://pub.dev/packages/latlong2).
- **Animations:** [Lottie](https://pub.dev/packages/lottie), [Flutter Animate](https://pub.dev/packages/flutter_animate), and [Animated Text Kit](https://pub.dev/packages/animated_text_kit).
- **Fonts:** [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter).

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (v3.7.2 or higher)
- Android Studio / VS Code with Flutter extension
- An active Internet connection for map tiling and geolocation

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Muzaffar01/netscope-pro.git
   cd netscope
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the application:**
   ```bash
   flutter run
   ```
